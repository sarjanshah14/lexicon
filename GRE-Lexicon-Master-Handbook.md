## 1. How to Use This Handbook

This is **not** a list to read front-to-back. It is a layered system. The key idea: **don't memorize words one at a time — learn them in families** (by root and by meaning), because that is also how the GRE tests them.

### How to learn a single word (the 4-step process)
Whenever you meet a new word, do these four things *in order*. This is the core habit of the whole handbook:

1. **Meaning** — read the short definition. Put it in your own words.
2. **Root / hook** — attach it to its root or memory hook (§3). This is the glue; a word with a hook is ~3× more likely to stick.
3. **Family** — find its synonyms and antonyms (§4, §8, §11). Never learn a word alone — learn "this word means X, *like* these, *unlike* those."
4. **Use** — read the example sentence and picture the blank it would fill. Words you can *use* are words you'll recognize on the test.

Then **test** it later from memory (§9) — recall, don't re-read.

### Which section to use, and when
| Section | What it's for | When to use it |
|---|---|---|
| **§2 High-Priority** | the 250 words that matter most | start here if you're short on time |
| **§3 Roots** | learn words in root-families | **start every study block here** |
| **§4 Meaning clusters** | tell close synonyms apart | after roots — this is the GRE's core skill |
| **§5 SE pairs** | ready-made Sentence-Equivalence answer pairs | when practicing SE questions |
| **§6 Polarity map** | positive / negative / neutral logic | when practicing Text Completion |
| **§7 Confusing pairs** | deliberate look-alike traps | drill these separately; they're easy points |
| **§8 Word groups** | every thematic group, fully listed | for broad coverage / second pass |
| **§9 Flashcards** | active-recall self-testing | every day, on words already learned |
| **§10 Revision plan** | the spaced schedule | to plan *when* to review |
| **§11 Glossary** | A–Z lookup of all 775 words | to check "what did that mean again?" |

### Two rules that matter most
- **Star words first.** When time is short, drill only the `*` high-frequency words — highest score per minute.
- **Space your reviews.** Re-test each batch on Day 1, 2, 4, 7, 14, 30 (§10). Spacing beats cramming for the long-term memory the GRE needs.

**Daily loop (~30 min):** (1) one new root family (§3) → (2) its matching meaning cluster (§4) → (3) 10 flashcards (§9) → (4) re-test yesterday's words.

## 3. Root Word Encyclopedia

The single highest-leverage way to memorize. Learn the root once; every member of its family becomes a derivation, not a separate fact. Only **defensible** roots are used; uncertain ones are flagged **(possible root)**.

### `bene-` / `bon-` = good, well
**Hook:** a *bene*fit is a *good* thing; *bon* appétit.
- **benevolence*** (n) — kindliness, charitableness (*bene* good + *vol* wish: "wishing well"). syn: altruism, humaneness → cruelty, misanthropy. *HF.* — "The benevolence of nuns caring for the homeless."
- **beneficence*** (n) — goodness, kindness in action. syn: charity, generosity → meanness, stinginess. *HF.*
- **beneficial** (adj) — helpful. syn: advantageous, salutary → harmful, detrimental.
- **benediction** (n) — a blessing (*bene* good + *dict* speak: "speaking well over"). syn: blessing, sanction → curse, malediction.
- **benign*** (adj) — favorable, mild; not harmful. *HF.* → malignant. **Family contrast:** *benevolent* = wishing good; *beneficent* = doing good; *benign* = harmless/gentle.

### `mal-` = bad, evil, ill — *the dark mirror of `bene-`*
**Hook:** *mal*function = works *badly*.
- **malevolent*** (adj) — wishing evil (*mal* bad + *vol* wish). syn: malicious, hostile → benign. *HF.* Direct antonym of *benevolent*.
- **malediction** (n) — a curse (*mal* + *dict* speak). syn: imprecation, execration → benediction.
- **malign** (v) — to slander, speak evil of. syn: vilify, defame → praise.
- **malignant** (adj, implied) — actively evil/harmful → benign.
- **maladroit** (adj) — clumsy, unskillful (*mal* badly + *adroit* skilled). syn: bungling, awkward → adroit, dexterous.
- **malfeasance / improbity** (Word-Groups: immoral) — wrongdoing. **Nuance:** *malevolent* = the wish; *malediction* = the spoken curse; *malign* = the act of slandering.

### `dict-` / `dic-` = to speak, say
**Hook:** a *dict*ionary tells you what words *say*.
- **benediction** / **malediction** — see above (blessing vs. curse).
- **didactic** (adj) — meant to teach or moralize (Gk *didak*, teach — sibling sense). syn: instructive, pedantic.
- **diction / denotation** (lexical) — the *denotation* is what a word literally says.
- **valedictory** (implied by *valediction* family) — a farewell (saying goodbye). **Use:** any *-dict-* word relates to **speech, pronouncement, or saying**.

### `loqu-` / `loc-` = to speak, talk
**Hook:** *elo*quent = speaks well.
- **loquacious*** (adj) — very talkative. syn: garrulous, gabby → taciturn, silent. *HF.*
- **circumlocution** (n) — roundabout speech (*circum* around + *loqu* speak). syn: verbosity, periphrasis → conciseness.
- **grandiloquent / magniloquent** (Word-Groups) — pompous, high-flown speech. **Contrast with `tac-` (silence) below.**

### `tac-` / `tic-` = silent
**Hook:** stay *tac*it = stay silent.
- **tacit*** (adj) — implied, unspoken. syn: implicit, inferred → explicit. *HF.*
- **taciturn** (adj) — habitually silent, reserved. syn: uncommunicative → talkative.
- **reticent*** (adj) — reserved, uncommunicative (*re-* + *tic*). syn: tight-lipped → unreserved. *HF.* **Family poles:** loquacious/garrulous (too much talk) ←→ taciturn/reticent/laconic (too little).

### `path-` = feeling, suffering
**Hook:** sym*path*y = feeling *with*.
- **apathy*** (n) — lack of feeling/interest (*a-* without). syn: indifference, unconcern → concern, interest. *HF.*
- **antipathy*** (n) — feeling against; intense dislike (*anti-* against). syn: aversion, animus → affection. *HF.*
- **empathy / sympathy** (implied family) — feeling with another.
- **impassive*** (adj, via `pass`) — showing no feeling (*im-* not + *pass* feeling). syn: stolid, unmoved → emotional. *HF.* **Spectrum:** apathy (none) → antipathy (negative feeling) → fervor/passion (intense feeling).

### `phil-` = love & `mis-` = hate (the love/hate axis)
**Hook:** *phil*osophy = love of wisdom; *mis*anthrope = hates people.
- **philanthropy / philanthropic** (implied; appears as antonym of *avarice*, *mendicant*) — love of humankind, charity. → misanthropy.
- **misanthrope*** (n) — a hater of humankind (*mis* hate + *anthrop* human). syn: cynic, recluse → optimist. *HF.*
- **misogynist** (n, listed as syn of misanthrope) — a hater of women.
- **xenophobe** (n) — one who fears/hates foreigners (*xeno* stranger + *phob* fear). **Pairing:** *anthrop* (human) appears in misanthrope and philanthropy — both about humankind, opposite valence.

### `fid-` = faith, trust
**Hook:** *fid*elity = faithfulness; Fido the loyal dog.
- **fidelity** (n) — faithfulness; accuracy. syn: loyalty, constancy → infidelity, disloyalty.
- **diffidence*** (n) — shyness; lack of self-trust (*dis-* away + *fid*). syn: timidity, shyness → self-assurance. *HF.*
- **perfidious*** (adj) — treacherous, faithless (*per-* through/betray + *fid*). syn: disloyal → loyal. *HF.*
- **confide / confidant** (implied) — to trust with secrets. **Note:** *fealty* (Word-Groups: loyalty) is a cousin.

### `cred-` = believe, trust
**Hook:** *cred*ibility = believability; in*cred*ible = un-believable.
- **credible** (adj) — believable, reliable. syn: plausible, dependable → implausible.
- **credulous** (adj, in confusing-words) — apt to believe anything (gullible). → skeptical.
- **creditable** (adj, in confusing-words) — deserving credit/esteem.
- **incredulous** (adj) — skeptical, disbelieving. syn: dubious, mistrustful → trusting.
- **credo / credence** (n) — a set of beliefs; belief. **CP trio:** *credible* (believable) / *credulous* (too believing) / *creditable* (praiseworthy) — see §7.

### `ver-` = truth
**Hook:** *ver*ify = make true; *ver*dict = "truth spoken."
- **veracious*** (adj) — truthful, accurate (*ver* truth). → evasive, vague. *HF.* **CP:** ≠ *voracious* (greedy).
- **veracity** (n) — truthfulness. → mendacity. **CP:** ≠ *voracity* (greed).
- **aver*** (v) — to declare to be true (*a-* + *ver*). syn: assert, avow → deny. *HF.*
- **verisimilar / verisimilitude** (adj/n) — appearing true. syn: credible, plausible.
- **veritable** (adj) — genuine, true. syn: authentic, bona fide → bogus.
- **verdant** (adj) — *false friend:* this is *ver* = green (vegetation), not truth — green/lush, or inexperienced.

### `mend-` / `menda-` = fault, lie (vs. `mend-` = beg)
**Hook:** *mend*acious tells lies; a beggar is *mend*icant — keep them apart.
- **mendacious*** (adj) — lying, deceitful. syn: deceitful, duplicitous → honest. *HF.*
- **mendacity** (n) — untruthfulness, lying. → veracity.
- **mendicant** (n) — *different sense:* a beggar (*mend* = beg). syn: panhandler → philanthropist. **CP:** *mendacity* (lying) vs. *mendicity* (begging) — §7.
- **emend / amend** (implied) — to correct a fault.

### `gen-` = birth, kind, origin
**Hook:** *gen*erate = bring into being; *gen*us = a kind.
- **engender*** (v) — to give rise to, produce (*en-* + *gen*). syn: produce, provoke → destroy. *HF.*
- **indigenous** (adj) — native, originating in a place. syn: native, aboriginal → alien, foreign.
- **generic** (adj) — general, of a whole class/kind. syn: collective → particular.
- **congenial** (adj) — agreeable, of kindred nature (*con-* together + *gen*). syn: kindred, sympathetic → disagreeable.
- **ingenuous** (adj) — innocent, frank, "born straight" → disingenuous, artful. **CP:** *ingenuous* (naive) vs. *ingenious* (clever) — §7.

### `greg-` = flock, herd
**Hook:** con*greg*ation = a gathered flock.
- **gregarious*** (adj) — sociable, fond of company (likes the herd). syn: sociable, friendly → unfriendly, antisocial. *HF.*
- **egregious** (adj) — conspicuously bad (*e-* out of + *greg*: "out of the flock," standing out). syn: flagrant, glaring → ordinary.
- **congregate / segregate** (implied) — gather into / set apart from the flock.

### `anim-` = mind, soul, spirit
**Hook:** *anim*ated = full of spirit/life.
- **magnanimity*** (n) — great-souled generosity (*magn* great + *anim* soul). syn: nobility, altruism → pettiness. *HF.*
- **equanimity** (n) — even-minded calm (*equ* even + *anim*). syn: composure, poise → agitation.
- **pusillanimous** (adj) — cowardly, "tiny-spirited" (*pusill* tiny + *anim*). syn: timid, craven → bold.
- **unanimous / animosity** (implied) — of one mind / hostility of spirit.

### `magn-` / `grand-` = great, large
**Hook:** *magn*ify, *grand*iose.
- **magnanimity*** — see above (great soul).
- **magnate** (n) — a great/powerful person. syn: tycoon, mogul → unknown.
- **magnum opus** (n) — a great work of art/literature.
- **aggrandizement*** (n) — increase of power/wealth/rank (*ad-* + *grand*). syn: advancement → degradation. *HF.*
- **grandiloquent / grandiose** (Word-Groups: show-off) — pompously grand.

### `voc-` / `vok-` = voice, call & `vol-` = wish/will
**Hook:** *voc*al = of the voice; in*vok*e = call upon.
- **equivocal*** (adj) — ambiguous, "equal voices" (*equi* equal + *voc*). syn: ambiguous, vague → unequivocal. *HF.*
- **vociferous** (adj, listed as syn) — loud, clamorous in voice.
- **advocate** (n, Word-Groups: supporter) — one who voices support.
- *(`vol-` = wish):* **benevolent / malevolent** — well-wishing / evil-wishing; **volition** (implied) = act of will. **Keep apart:** *voc* (voice) vs. *vol* (will) — both appear in *equivocal* vs. *benevolent*.

### `dur-` = hard, lasting
**Hook:** *dur*able = lasts; en*dur*e = stay hard through.
- **obdurate*** (adj) — stubborn, hardened (*ob-* against + *dur* hard). syn: inflexible, hardhearted → compassionate. *HF.*
- **durable / endure** (Word-Groups: determined) — lasting; to last.
- **duress** (n) — coercion by force/threat. syn: constraint, pressure → free will.

### `plac-` = please, calm
**Hook:** *plac*id water is calm; *plac*ate = make calm.
- **placate*** (v) — to appease, calm. syn: mollify, pacify → antagonize. *HF.*
- **implacable*** (adj) — that cannot be appeased (*im-* not + *plac*). syn: relentless, unforgiving → lenient. *HF.*
- **placid** (adj, Word-Groups: calm) — peacefully calm. syn: serene, tranquil → turbulent.
- **complacent** (adj) — self-satisfied (pleased with oneself). → insecure. **CP:** *complacent* (smug) vs. *complaisant* (eager to please) — §7.

### `plic-` / `plex-` = fold
**Hook:** im*plic*it = "folded in" / implied; com*plex* = folded together.
- **implicit*** (adj) — implied, not stated; also unreserved. syn: tacit, inferred → explicit. *HF.*
- **implicate / inculpate** — to fold someone into a crime; blame.
- **explicit** (adj, antonym anchor) — fully unfolded, stated plainly.

### `tract-` = pull, draw, handle
**Hook:** a *tract*or pulls; trac*table* can be handled.
- **tractable*** (adj) — easily managed, "handle-able." syn: docile, amenable → obstinate. *HF.*
- **intractable*** (adj) — stubborn, unmanageable (*in-* not). syn: unruly, obstinate → docile. *HF.* (Direct antonym pair.)
- **protract** (v) — to draw out in time. syn: extend, prolong → curtail.
- **detract / distract** (implied) — pull away from. **Twin pair to memorize together:** *tractable* ↔ *intractable*.

### `tort-` = twist
**Hook:** con*tort* = twist up; ex*tort* = twist money out.
- **tortuous*** (adj) — full of twists; complex. syn: winding, circuitous → straight. *HF.* **CP:** ≠ *torturous* (painful).
- **distort / contort / extort** (implied) — twist out of shape / twist out by force.

### `fus-` / `fund-` = pour, melt
**Hook:** in*fus*e = pour in; *fund* = the bottom poured into.
- **diffuse*** (adj/v) — spread/poured out widely; wordy (*dif-* apart + *fus*). syn: scattered, verbose → concentrated, concise. *HF.*
- **effusive** (adj) — gushing, overflowing with feeling (*ex-* out + *fus*). syn: gushing, profuse → reserved.
- **profundity*** (n) — great depth (*pro-* + *fund* bottom). → shallowness. *HF.*
- **suffuse / infuse** (implied) — pour over / pour into.

### `son-` / `phon-` = sound
**Hook:** *son*ic, tele*phon*e.
- **cacophony*** (n) — harsh discordant sound (*caco* bad + *phon*). syn: dissonance, din → harmony. *HF.*
- **euphonious** (adj) — sweet-sounding (*eu* good + *phon*). syn: melodious, mellifluous → cacophonous.
- **dissonance / consonance** (implied) — clashing/agreeing sound. **eu-/caco- contrast:** good sound vs. bad sound.

### `eu-` = good, well & `caco-` = bad (Greek prefixes)
- **eulogy*** (n) — speech of high praise (*eu* good + *log* speech). → diatribe. *HF.*
- **euphonious** — sweet-sounding (see above).
- **euphoria** (Word-Groups: happy) — a feeling of intense well-being.
- **cacophony*** — harsh sound (see above). **Mnemonic:** *eu-* always positive, *caco-* always negative.

### `log-` / `-logy` = speech, word, study
**Hook:** dia*log*ue, eu*log*y.
- **eulogy*** — high praise (good speech).
- **analogy*** (n) — a comparison of likeness (*ana-* + *log* ratio). syn: parallel → difference. *HF.*
- **prologue / epilogue / neologism** (implied) — fore-speech / after-speech / new word.

### `spec-` / `spic-` = look, see
**Hook:** in*spec*t, *spec*tacles.
- **perspicacious** (adj) — keenly seeing/perceptive (*per-* through + *spic*). syn: astute, shrewd → undiscerning.
- **circumspect** (adj, Word-Groups: cautious) — looking around carefully; prudent.
- **specious*** (adj) — looks good but is false (*spec* look). syn: spurious → straightforward. *HF.*
- **ostensible*** (adj, via *tend*) — what is "shown"/apparent. **CP:** *perspicacious* (perceptive) vs. *perspicuous* (clearly expressed) — §7.

### `scrib-` / `script-` = write
**Hook:** *scrib*ble, manu*script*.
- **proscribe*** (v) — to forbid (*pro-* + *scrib*: "write off"/outlaw). syn: prohibit, ban → permit. *HF.* **CP:** ≠ *prescribe* (recommend) — §7.
- **circumscribe** (implied) — to draw a limiting line around.
- **ascribe / inscribe** (implied) — to attribute / to write into.

### `cur-` / `curs-` = run
**Hook:** *cur*rent, *curs*or runs across the screen.
- **precursor*** (n) — a forerunner (*prae-* before + *curs* run). syn: predecessor, herald → successor. *HF.*
- **cursory** (adj, Word-Groups: not thorough) — hasty, "running over" the surface. syn: perfunctory, superficial.
- **discursive / incursion** (implied) — rambling / a running-in/raid.

### `sed-` / `sid-` / `sess-` = sit, settle
**Hook:** *sed*entary = sitting a lot; re*sid*e = settle in.
- **assiduous*** (adj) — diligent, "sitting at" a task (*ad-* + *sid*). syn: industrious, sedulous → lazy. *HF.*
- **sedulous** (adj) — persistent, painstaking. syn: diligent, assiduous → negligent.
- **subside*** (v) — to settle down, abate (*sub-* + *sid*). syn: abate, wane → surge. *HF.*
- **sedate / sedentary / insidious** (implied) — calm / sitting / lying in wait.

### `flu-` / `flux-` = flow
**Hook:** *flu*id, *flu*ent.
- **affluence** (n) — abundant wealth ("flowing to"). syn: opulence, riches → poverty.
- **mellifluous** (adj) — sweetly flowing (*mel* honey + *flu*). syn: dulcet, melodious → harsh.
- **superfluous** (adj) — overflowing, more than needed (*super-* + *flu*). syn: redundant, surplus → essential.
- **confluence / fluctuate** (implied) — flowing together / flowing up and down.

### `volv-` / `volu-` = roll, turn & `vol-` = fly
- **convoluted** (implied) — rolled together, complex.
- **voluble** (adj, Word-Groups: talkative) — words "rolling out" fluently.
- **volatile*** (adj) — *(`vol` = fly)* tending to fly off; unstable, explosive. syn: mercurial, capricious → constant. *HF.* (Note: different root from "roll.")

### `vit-` / `viv-` / `via` = life, way
**Hook:** *vit*al, *viv*id, sur*viv*e.
- **viable*** (adj) — workable; able to live (*via* way/life). syn: feasible, workable → unworkable. *HF.*
- **vivacious** (adj, syn of ebullient) — lively. → listless.
- **convivial** (adj) — sociable, festive (*con-* + *viv*: "living together"). syn: jovial, sociable → cold, gloomy.
- **revive / vivify** (implied) — bring back to life.

### `mort-` = death
**Hook:** *mort*al, *mort*uary.
- **moribund** (adj) — dying, near death. syn: declining, expiring → flourishing.
- **mortify** (v) — to humiliate ("kill" with shame); to discipline the flesh. syn: embarrass, humiliate → delight.
- **immortal / mortician** (implied) — undying / death-handler.

### `nov-` / `neo-` = new
**Hook:** *nov*el, in*nov*ate.
- **novice / neophyte** (n, syn of *tyro*) — a beginner ("the new one").
- **neologism** (implied) — a new word.
- **renovate / innovate** (implied) — make new again / introduce new.

### `arch-` = chief, rule, first; `-cracy` / `-archy` = rule
**Hook:** mon*arch* = sole ruler; *arch*etype = first model.
- **archetype** (n) — the original model (*arch* first + *typ* pattern). syn: prototype, ideal.
- **archaic** (adj) — belonging to an earlier time, old-fashioned (*arch* first/old). syn: obsolete, antiquated → modern.
- **autarchy** (n) — absolute rule (*auto* self + *arch* rule). syn: autocracy, despotism. (≠ *autarky* = economic self-sufficiency.)
- **anarchy / hierarchy** (implied) — no rule / ranked rule.

### `chron-` = time
**Hook:** *chron*ology, syn*chron*ize.
- **anachronism** (n) — something out of its proper time (*ana-* against + *chron*).
- **synchronous** (adj) — occurring at the same time (*syn* together + *chron*). syn: concurrent, coincident → unrelated.
- **chronic** (implied) — lasting over time.

### Additional high-yield mini-roots

`path-`, `cred-`, and `dur-` are covered above. Here are six more roots worth knowing — **read one row at a time**: cover the right column, say the root's meaning, then check.

| Root | Meaning | Word family (with mini-hook) |
|---|---|---|
| `ambi-` / `amphi-` | both | **ambivalence\*** = both feelings · **ambiguous** = both meanings · **ambidextrous** = both hands |
| `circum-` | around | **circumlocution** = talking around · **circumspect** = looking around → cautious · **circumvent** = going around → evade · **circumscribe** = drawing a line around |
| `ob-` | against / toward (intensive) | **obdurate** = hardened against · **obstreperous** = noisy against · **obsequious** = following toward → fawning · **obviate** = meet in the way → prevent · **obfuscate** = darken · **obtrude** = thrust against |
| `per-` | through / thoroughly (or "wrongly") | **perspicacious** = seeing through · **pervade / pervasive** = going through · **perfidious** = faith-betrayal · **pernicious** = thoroughly destructive · **peruse** = read through |
| `equ-` / `equi-` | equal, even | **equanimity** = even mind · **equivocal** = equal voices · **equable / equitable** = even / fair · **iniquity** = un-evenness → wickedness |
| `sol-` | alone | **solitary / recluse** = alone · **solipsism** = self alone · **desolate** = left alone → bleak |

**Twin-antonym pairs to drill back-to-back.** These are opposites built on the roots above — learning each pair *together* makes both stick, and they show up constantly in Sentence Equivalence:

| Word | ↔ | Opposite |
|---|---|---|
| tractable (manageable) | ↔ | intractable (unmanageable) |
| placate (calm) | ↔ | provoke / implacable (cannot be calmed) |
| benevolent (good-willed) | ↔ | malevolent (ill-willed) |
| benediction (blessing) | ↔ | malediction (curse) |
| euphony (pleasant sound) | ↔ | cacophony (harsh sound) |
| eulogy (speech of praise) | ↔ | diatribe (speech of attack) |
| veracious (truthful) | ↔ | mendacious (lying) |

> **How to drill roots (do this daily):**
> 1. Pick **one** root box per day.
> 2. Cover the word list; read **only** the root and its meaning.
> 3. Try to *generate* the family yourself from the meaning — say each word and its hook aloud.
> 4. Uncover and check. Mark any you missed.
> 5. Learn antonym roots **as pairs** (`bene-/mal-`, `loqu-/tac-`, `eu-/caco-`) — each one reinforces the other.

## 4. Semantic / Synonym Clusters

**What this section is, in one line:** words grouped by *what they mean*, so you learn 10 related words as one idea instead of 10 separate facts.

**Why it matters:** the GRE almost never asks "what does X mean?" in isolation. It asks you to tell **close synonyms apart** — which "praise" word fits, which "criticism" word is too strong, which two words mean the *same* thing. That skill is what these clusters build.

**Each cluster has the same seven labels. Here is exactly what each one means:**

| Label | What it tells you | How to use it |
|---|---|---|
| **Core meaning** | the single idea all the words share | this is what a blank "pointing here" wants |
| **Words** | every member of the family | your raw vocabulary to learn |
| **Key differences** | the fine shades that separate near-synonyms | this decides the *right* answer vs. a close-but-wrong one |
| **Most useful** | the highest-frequency words to learn first | learn these before the rare ones |
| **Traps** | words that look like they belong but don't (or flip meaning) | the GRE plants these as wrong answers |
| **SE clue** | a sample Sentence-Equivalence sentence | shows how the idea appears on the test |
| **Hook** | a memory aid | makes the words stick |

**How to study one cluster (5 minutes):**
1. Read the **Core meaning** and the **Words**. Say each word + its meaning aloud.
2. Read **Key differences** twice — this is the part that earns points.
3. Cover the cluster and try to list 5+ words from the Core meaning alone.
4. Note the **Traps** so you can reject them on test day.
5. Come back tomorrow and re-test (see §10).

> **Note on cross-links:** a word listed in two clusters (e.g. *adulation* in both Praise and Flattery) is one the test loves to make ambiguous — pay attention to which clue word tips it one way or the other.

### 4.1 PRAISE / APPROVAL
- **Core meaning:** to express high approval; an expression of it.
- **Words:** acclaim, accolade, applaud, approbation, commendable, eulogy, exalt, extol, glorify, hail, laud, lionize, plaudit, adulation*, panegyric, encomium, tribute, laurels.
- **Key differences:** **laud / extol / acclaim** = praise highly (neutral-positive). **eulogy / panegyric / encomium** = a *formal speech/piece* of praise. **adulation** = *excessive*, fawning praise (leans negative). **lionize** = treat someone as a celebrity. **accolade / plaudit / laurels** = the *award/honor* itself.
- **Most useful:** **laud, extol, eulogy, adulation\*, approbation, acclaim**.
- **Traps:** **adulation** (too much praise — not neutral); **eulogy** (specifically a *speech*, often funeral) vs general praise.
- **SE clue:** "Critics could not stop ______ the film" → **lauding / extolling**.
- **Hook:** *Lions get LIONized; a EULOGY is a EU- (good) LOGY (speech).*

### 4.2 CRITICISM / SCOLD / VERBAL ATTACK
- **Core meaning:** to attack with words; harsh disapproval.
- **Words:** berate*, castigate*, censure, chide, excoriate, lambaste, rebuke, reprove, reproach, upbraid, vituperate, revile, fulminate, rail, rant, tirade, diatribe, harangue, invective, objurgate, decry, denounce, assail, disparage, deprecate.
- **Key differences:** **berate / upbraid / chide / reprove** = scold (mild→firm). **castigate / excoriate / lambaste / vituperate** = *severe, blistering* attack. **diatribe / tirade / harangue / invective** = the *long angry speech* (nouns). **disparage / decry / deprecate** = belittle / speak against (cooler tone).
- **Most useful:** **castigate\*, excoriate, berate\*, censure, vituperate, diatribe, harangue**.
- **Traps:** **deprecate** ≠ depreciate (lose value); **disparage** is milder than **excoriate**; a **diatribe** is a *thing said*, not the act.
- **SE clue:** "The review was a ______ of everything the author stood for" → **diatribe / tirade**.
- **Hook:** *exCORIate = strip the CORTEX (skin) off with words.*

### 4.3 FLATTERY / SERVILE PRAISE
- **Core meaning:** insincere praise to win favour.
- **Words:** fawn*, fulsome, obsequious, sycophant, blandishment*, cajole, wheedle, palaver, toady, adulation*.
- **Key differences:** **obsequious / sycophant(ic) / toady** = servile, boot-licking. **fawn** = behave servilely. **fulsome** = praise so excessive it disgusts. **blandishment / cajole / wheedle** = coaxing flattery to *get something*.
- **Most useful:** **obsequious, sycophant, fawn\*, fulsome, cajole**.
- **Traps:** **fulsome** is *negative* (overdone), not "full and generous" — a classic GRE trap. Links to [praise] but with a sour edge.
- **SE clue:** "His ______ compliments embarrassed everyone" → **obsequious / fulsome**.
- **Hook:** *A SYCOphant is SICK with flattery; OBSEQUIous = following OB- closely.*

### 4.4 DECEPTION / TRICKERY / FALSEHOOD
- **Core meaning:** to mislead; the quality of being false.
- **Words:** artifice, artful, beguile, bilk, chicanery, crafty, duplicity, dupe, fleece, guile, hoodwink, mislead, swindle, wily, dissemble, dissimulate, feign, mendacity, perjury, prevarication, sophistry, specious, spurious, ersatz, apocryphal*, fabricate, fallacious, sham, phony, subterfuge.
- **Key differences:** **guile / duplicity / chicanery / artifice** = the *quality/practice* of deceit. **dissemble / dissimulate / feign** = to *hide true feelings*. **mendacity / prevarication / perjury** = *lying* (perjury = under oath). **specious / spurious / fallacious / sophistry** = *false reasoning that looks valid*. **ersatz / sham / phony / apocryphal** = fake / of doubtful authenticity.
- **Most useful:** **specious, mendacity, duplicity, guile, dissemble, chicanery, apocryphal\***.
- **Traps:** **specious** = *plausible but false* (not merely "spacious"!); **ingenuous** (naive) ≠ **ingenious** (clever) — see §7.
- **SE clue:** "The argument was ______ — convincing at first, but baseless" → **specious / fallacious**.
- **Hook:** *SPECIOUS = good-looking SPECImen, rotten inside; DUPLICity = DOUBLE-dealing.*

### 4.5 TRUTH / HONESTY / FRANKNESS
- **Core meaning:** truthful, upright, straightforward.
- **Words:** veracity, veracious, probity, rectitude, integrity, candor, frank, forthright, ingenuous, guileless, artless, aboveboard.
- **Key differences:** **veracity** = truthfulness; **probity / rectitude / integrity** = moral *uprightness*; **candor / forthright / frank** = *plain-spokenness*; **ingenuous / guileless / artless** = *innocent, unguarded* honesty.
- **Most useful:** **veracity, probity, candor, ingenuous, integrity**.
- **Traps:** **ingenuous** (honest/naive) is the *opposite* of disingenuous; do not confuse with **ingenious**. **candor** ≠ "candor as kindness" — it means bluntness.
- **SE clue:** "Known for her ______, she never softened the truth" → **candor / frankness**.
- **Hook:** *VERacity = VERity = truth; PROBity is PROVed character.*

### 4.6 INTELLIGENCE / INSIGHT / SHREWDNESS
- **Core meaning:** mentally sharp; perceptive.
- **Words:** acuity, acumen, astute, discerning, discriminating, insightful, judicious, perceptive, perspicacious, prudent, sagacious, sage, savant, erudite*, adroit, cogent.
- **Key differences:** **acumen / acuity** = sharpness of mind (nouns). **astute / shrewd / perspicacious / sagacious** = penetrating judgment of people/situations. **judicious / prudent** = *wise & cautious* in decisions. **erudite** = deeply *learned* (book-knowledge), not necessarily street-smart. **discerning / discriminating** = good at telling fine things apart.
- **Most useful:** **perspicacious, sagacious, acumen, astute, erudite\*, judicious**.
- **Traps:** **discriminating** here = *having good taste* (positive), not "biased." **erudite** ≠ shrewd.
- **SE clue:** "A ______ investor, she foresaw the crash" → **perspicacious / sagacious / astute**.
- **Hook:** *PERSPICACIOUS = sees PER (through); SAGE/SAGACIOUS = wise old sage.*

### 4.7 STUPIDITY / FOOLISHNESS
- **Core meaning:** lacking sense.
- **Words:** asinine, fatuous*, imbecile, inane, oafish, obtuse, vacuous, vapid, callow, puerile.
- **Key differences:** **fatuous / asinine / inane / vacuous / vapid** = silly, empty, pointless. **obtuse** = *slow to understand*. **oafish** = clumsily stupid. **callow / puerile** = *immature* (young-foolish).
- **Most useful:** **fatuous\*, inane, obtuse, vacuous, callow**.
- **Traps:** **obtuse** (dull-witted) vs its geometry meaning; **vapid/vacuous** = empty, not merely boring.
- **SE clue:** "His ______ grin showed he understood nothing" → **fatuous / vacuous**.
- **Hook:** *ASININE = like an ASS; obTUSE = mentally a blunt (obtuse) angle.*

### 4.8 CALM / COMPOSURE  ↔  AGITATION
- **Core meaning (calm):** unshaken, serene.
- **Words (calm):** aplomb*, composure, equanimity, equable, imperturbable, impassive, nonchalant, phlegmatic, placid, serene, staid, stolid, tranquil, sangfroid.
- **Words (agitate / disturb):** consternation, disconcert, dismay, disquiet, distraught, faze, fluster, foment, frenetic, overwrought, perturb, agitate, roil.
- **Key differences:** **equanimity / aplomb / composure / sangfroid** = *self-possession under pressure*. **phlegmatic / stolid / impassive / staid** = *temperamentally* unemotional. **placid / serene / tranquil** = peaceful in quality. On the other side, **foment** = *stir up* trouble (active), **perturb / disquiet / faze** = unsettle.
- **Most useful:** **equanimity, aplomb\*, imperturbable, phlegmatic, foment, perturb, disconcert**.
- **Traps:** **foment** = incite (causes agitation) — opposite polarity from the calm list though same field; **staid** ≠ "stayed."
- **SE clue:** "She met the crisis with ______, never raising her voice" → **equanimity / aplomb**.
- **Hook:** *EQUANIMITY = EQUal mind; PHLEGMatic = full of slow "phlegm."*

### 4.9 INCREASE / GROW  ↔  DECREASE / WEAKEN
- **Core meaning (increase):** to grow, strengthen, intensify.
- **Words (up):** augment, burgeon*, accrue, amplitude, proliferate, wax, bolster*, buttress*, fortify, foster, kindle.
- **Words (down/weaken):** abate, attenuate, atrophy, attrition, debilitate, enervate, enfeeble, sap, wane, wilt, wither, flag, diminish, dwindle, ebb, subside, mitigate, assuage, allay.
- **Key differences:** **burgeon / proliferate / wax** = grow rapidly. **augment / bolster / buttress / fortify** = *add strength/support*. On the down side, **attenuate / abate / wane / subside / ebb** = lessen gradually; **enervate / debilitate / sap / atrophy** = *drain strength* (enervate = weaken, NOT "energize"!).
- **Most useful:** **burgeon\*, augment, bolster\*, attenuate, enervate, abate, mitigate**.
- **Traps:** **enervate** is the #1 trap — it means *weaken*, the opposite of "energize." **mitigate / assuage / allay / abate** cluster with [calm].
- **SE clue:** "The long march ______ the troops, leaving them too weak to fight" → **enervated / debilitated**.
- **Hook:** *eNERVate cuts your NERVE/energy; BURGEON = a bud (burgeon) bursting open.*

### 4.10 SUPPORT / DEFEND  ↔  ATTACK / OPPOSE
- **Core meaning (support):** to back, uphold, confirm.
- **Words (support):** advocate, ally, champion, endorse, espouse, proponent, tout, bolster*, buttress*, corroborate, substantiate, validate, vindicate, ratify, sanction, affirm.
- **Words (attack/oppose):** adversary, antagonist, detractor, impugn, inimical, nemesis, assail, belie, debunk, discredit, refute, rebut, gainsay, controvert, confute.
- **Key differences:** **advocate / champion / espouse / proponent** = *back a cause*. **corroborate / substantiate / validate / vindicate** = *back a claim with evidence*. To attack a claim: **refute / rebut / confute / controvert / gainsay** (gainsay = deny). **belie / debunk / discredit** = *expose as false*. **impugn** = *call into question*.
- **Most useful:** **corroborate, substantiate, refute, impugn, espouse, vindicate, debunk**.
- **Traps:** **sanction** = either *approve* OR *penalize* (auto-antonym — context decides). **belie** = *contradict/disguise*, not "to lie."
- **SE clue:** "New data ______ the theory, confirming its central claim" → **corroborated / substantiated**.
- **Hook:** *imPUGN = PUGilist punching your argument; corROBORate = re-en-FORCE (robur = strength).*

### 4.11 WORDY / TALKATIVE  ↔  TERSE / SILENT
- **Core meaning (wordy):** using too many words.
- **Words (wordy):** garrulous, loquacious, prolix, verbose, voluble, long-winded, verbiage, grandiloquent, bombastic, turgid, magniloquent.
- **Words (terse/quiet):** laconic*, terse, succinct, pithy, concise, curt, brusque*, taciturn, reticent, reserved, monosyllabic.
- **Key differences:** **garrulous / loquacious / voluble** = *chatty* (about a person). **prolix / verbose / long-winded** = *tediously wordy* (about text/speech). **grandiloquent / bombastic / turgid / magniloquent** = *pompously* wordy. Quiet side: **laconic / terse / pithy / succinct** = *briefly effective*; **taciturn / reticent / reserved** = *disinclined to talk* (personality).
- **Most useful:** **garrulous, loquacious, prolix, laconic\*, taciturn, reticent, succinct, bombastic**.
- **Traps:** **taciturn** (silent by nature) ≠ **reticent** (reluctant to *reveal*); **laconic** ≠ rude.
- **SE clue:** "Famously ______, he answered every question with a single word" → **laconic / taciturn**.
- **Hook:** *LACONIC = the LACONians (Spartans) of few words; GARRULOUS gargles endlessly.*

### 4.12 CLARITY  ↔  OBSCURITY / VAGUENESS
- **Core meaning (clear):** easy to understand.
- **Words (clear):** lucid, limpid, pellucid, perspicuous, clarity, elucidate, expound, exposition, cogent, intelligible.
- **Words (obscure):** abstruse*, arcane, recondite, esoteric*, occult, cryptic, opaque, turbid, nebulous, amorphous, inscrutable, enigmatic, obfuscate, becloud, murky.
- **Key differences:** **lucid / limpid / pellucid / perspicuous** = transparently clear. **elucidate / expound** = to *make clear*. Obscure side: **abstruse / recondite / esoteric / arcane** = *hard because deep or specialized*; **cryptic / enigmatic / inscrutable** = *mysterious*; **opaque / turbid / nebulous / murky** = *unclear in quality*; **obfuscate / becloud** = *to make unclear*.
- **Most useful:** **lucid, perspicuous, elucidate, abstruse\*, recondite, esoteric\*, obfuscate, opaque**.
- **Traps:** **perspicuous** (clear writing) vs **perspicacious** (clear *seeing* = shrewd) — see §7. **specious** is not here (it's [deception]).
- **SE clue:** "The manual was so ______ that even novices understood it" → **lucid / perspicuous**.
- **Hook:** *peLLUCID lets LIGHT through; obFUSCate = make FUSCous (dark/murky).*

### 4.13 CERTAINTY  ↔  AMBIGUITY / DOUBT
- **Core meaning (certain):** definite, beyond question.
- **Words (certain):** absolute, categorical, certitude, conviction, emphatic, incontrovertible, unequivocal, unqualified, dogmatic.
- **Words (ambiguous/doubtful):** ambivalent, ambiguous, equivocal*, dubious, tentative, contingent, provisional, qualify, indeterminate, nebulous, vacillate, waver.
- **Key differences:** **categorical / unequivocal / incontrovertible / unqualified** = *admitting no doubt*. **certitude / conviction** = inner sureness. Doubt side: **equivocal / ambiguous** = *open to >1 reading*; **ambivalent** = *having mixed feelings* (people), not the same as ambiguous (statements); **tentative / provisional / contingent** = *not final*; **vacillate / waver** = *fail to decide*.
- **Most useful:** **unequivocal, categorical, equivocal\*, ambivalent, tentative, vacillate**.
- **Traps:** **ambiguous** (a *statement* with 2 meanings) vs **ambivalent** (a *person* with 2 feelings) — classic GRE split. **equivocal** can imply *deliberately* misleading.
- **SE clue:** "His ______ reply could be read as either yes or no" → **equivocal / ambiguous**.
- **Hook:** *unEQUIVOCAL = one EQUAL VOICE only; ambiVALENT = both (ambi-) ways pulled.*

### 4.14 WEALTH / ABUNDANCE  ↔  POVERTY / STINGINESS
- **Words (wealth/abundance):** affluence, opulence, prosperity, lavish, profuse, plethora, surfeit, copious, myriad, teeming.
- **Words (poverty):** penury, penurious, destitute, impecunious, indigent, pauper.
- **Words (stingy / unwilling to spend):** avarice, miserly, niggardly, parsimonious, penurious, skinflint, stingy, tightfisted, frugal, abstemious*, cupidity, covetous, rapacious.
- **Key differences:** **affluence / opulence** = rich(ness); **plethora / surfeit / glut** = *excess* (often too much). Poverty: **penury / destitution / indigence / impecunious**. Stinginess: **parsimonious / niggardly / miserly / penurious** = unwilling to spend; **avarice / cupidity / rapacious / covetous** = *greedy to acquire*. **frugal / abstemious** = *thrifty in a virtuous way*.
- **Most useful:** **penury, parsimonious, miserly, avarice, opulence, surfeit, indigent, abstemious\***.
- **Traps:** **frugal/abstemious** (positive thrift) vs **miserly/niggardly** (negative cheapness); **surfeit/plethora** = too much, not just "a lot."
- **SE clue:** "Despite his ______, he refused to buy even a winter coat" → **parsimony / miserliness**.
- **Hook:** *PENURY = empty PENNY purse; AVARICE = "have-a-rich" greed.*

### 4.15 KINDNESS / GENEROSITY  ↔  CRUELTY / MALICE
- **Words (kind):** benevolence*, beneficence*, benediction, benign*, altruism, philanthropy, magnanimous, clemency, largesse, munificent.
- **Words (cruel/malicious):** malevolence, malice, malign, maleficent, baleful, baneful, pernicious, rancor, vindictive, truculent, callous, heinous.
- **Key differences:** **benevolence / beneficence / altruism** = active goodwill; **magnanimous / munificent / largesse** = *generous & big-hearted*; **clemency** = mercy. Cruel side: **malevolence / malice / rancor** = ill-will; **malign / maleficent / baleful / baneful / pernicious** = *actively harmful*; **vindictive** = vengeful.
- **Most useful:** **benevolence\*, magnanimous, munificent, malevolence, pernicious, rancor, vindictive**.
- **Traps:** **magnanimous** = *forgiving/noble*, not just "generous"; **benign** can mean harmless (medical) or kindly.
- **SE clue:** "A ______ benefactor, she funded the entire hospital" → **munificent / magnanimous**.
- **Hook:** *BENE = good, MAL = bad — the master antonym pair (see §3).*

### 4.16 STUBBORN / INFLEXIBLE  ↔  YIELDING / FLEXIBLE
- **Words (stubborn):** adamant, obdurate, obstinate, intractable, intransigent, recalcitrant, refractory, pertinacious, perverse, willful, dogged, inexorable, implacable.
- **Words (yielding):** amenable, tractable, malleable, pliable, docile, compliant, acquiescent*, supple, complaisant.
- **Key differences:** **obdurate / obstinate / adamant** = won't change mind; **intractable / recalcitrant / refractory / intransigent** = *resistant to control/compromise*; **inexorable / implacable** = *cannot be moved or appeased*; **pertinacious / dogged** = *persistently* holding on. Yielding: **amenable / tractable / docile / compliant / acquiescent** = easily led; **malleable / pliable / supple** = *shapeable*.
- **Most useful:** **intransigent, recalcitrant, obdurate, intractable, amenable, tractable, malleable, acquiescent\***.
- **Traps:** **implacable** = can't be *appeased* (links to [calm/agitation]); **tractable ↔ intractable** is a frequent SE pair.
- **SE clue:** "Negotiations failed because both sides were ______" → **intransigent / obdurate**.
- **Hook:** *inTRACTable = can't be TRACTored/dragged; reCALCITRant = kicks back (calx = heel).*

### 4.17 PRIDE / ARROGANCE  ↔  HUMILITY
- **Words (arrogant):** arrogant, haughty, supercilious, imperious, disdainful, pompous, hubris, vainglorious, conceited, insolent, overweening, cavalier.
- **Words (humble):** humble, modest, unassuming, diffident, demure, meek, deferential.
- **Key differences:** **haughty / supercilious / disdainful** = *looking down* on others; **imperious / overweening** = *domineering*; **hubris** = *excessive pride that invites downfall*; **pompous / vainglorious / conceited** = self-important. Humble side: **diffident** = *shy/lacking confidence* (not just modest); **deferential** = yielding respectfully.
- **Most useful:** **supercilious, haughty, imperious, hubris, diffident, deferential**.
- **Traps:** **diffident** ≠ "indifferent"; **cavalier** = *haughtily dismissive*, not "gallant" here.
- **SE clue:** "Her ______ manner suggested she thought everyone beneath her" → **supercilious / haughty**.
- **Hook:** *superCILIous = raised eyebrow (cilia); HUBRIS = the hero's fatal pride.*

### 4.18 COURAGE  ↔  COWARDICE / FEAR
- **Words (brave):** audacious, dauntless, intrepid, valiant, valor, gallant, plucky, undaunted, temerity, chivalry.
- **Words (fearful/cowardly):** craven, pusillanimous, timorous, cower, cringe, flinch, trepidation, daunt, dastardly.
- **Key differences:** **intrepid / dauntless / undaunted / valiant** = fearless; **audacious / temerity** = *bold to the point of recklessness/rudeness*; **plucky / gallant** = spirited. Fear: **craven / pusillanimous / dastardly** = cowardly; **timorous / trepidation** = *timid/anxious*; **cower / cringe / flinch** = shrink back.
- **Most useful:** **intrepid, audacious, temerity, craven, pusillanimous, trepidation**.
- **Traps:** **audacity/temerity** carry a *negative* edge (nerve, gall) as well as "courage"; **pusillanimous** = literally "small-spirited."
- **SE clue:** "He lacked the courage to object — a truly ______ silence" → **craven / pusillanimous**.
- **Hook:** *inTREPID = no TREPIDation; PUSILLanimous = puny (pusillus) soul.*

### 4.19 PRAISE-WORTHY ORDER  ↔  CHAOS / CONFUSION
- **Words (chaos/confuse):** bedlam, pandemonium, chaos, tumult, melee, baffle, bewilder, confound, perplex, befuddle, mystify, obfuscate, discombobulate.
- **Words (puzzling things):** conundrum, enigma, cryptic, inscrutable, labyrinth, maze, intrigue, garble.
- **Key differences:** **bedlam / pandemonium / tumult / melee** = *noisy disorder*; **baffle / confound / perplex / mystify / bewilder** = *to puzzle a person*; **conundrum / enigma** = *the puzzle itself*; **garble** = *jumble a message*.
- **Most useful:** **pandemonium, confound, enigma, inscrutable, conundrum**.
- **Traps:** **confound** also = *mix up / refute*; an **enigma** is the puzzle, **perplex** is the effect on you.
- **SE clue:** "The contradictory clues only ______ the detective" → **baffled / confounded**.
- **Hook:** *BEDLAM = old London madhouse; ENIGMA = the unsolved code.*

### 4.20 SHORT-LIVED  ↔  LASTING
- **Words (fleeting):** ephemeral*, evanescent, transient, transitory, fleeting, momentary, fugitive.
- **Words (lasting):** enduring, permanent, perennial, abiding, immutable, indelible, perpetual.
- **Key differences:** **ephemeral / evanescent** = lasting a *very* short time (a day, a moment); **transient / transitory / fleeting** = passing; **fugitive** = fleeting/elusive. Lasting: **immutable** = *unchangeable*; **indelible** = *cannot be erased*; **perennial / perpetual / abiding** = ongoing.
- **Most useful:** **ephemeral\*, evanescent, transient, immutable, indelible**.
- **Traps:** **ephemeral ↔ enduring/permanent** is a top SE antonym set; **immutable** = can't change (not "can't move").
- **SE clue:** "Fame proved ______ — gone within a season" → **ephemeral / fleeting / transient**.
- **Hook:** *EPHEMERAL = lasts one day (Greek hēmera = day); eVANESCent = VANISHes.*

> **Cross-cluster map:** **adulation** sits in [praise] *and* [flattery]; **sanction / belie** are auto-antonyms in [support/attack]; **foment / implacable** bridge [calm↔agitation] and [stubborn]; **mitigate / assuage / allay / abate** appear in both [decrease] and [calm]. When a word lives in two clusters, learn *which clue word* tips it one way — that disambiguation is exactly what SE rewards.

## 5. Sentence Equivalence Clusters

> **What SE actually tests:** you pick **two** words that (a) both fit the sentence and (b) produce **the same meaning**. So the answer is always a **synonym pair** hidden among distractors — some of which are synonyms of *each other* but wrong for the sentence (the classic "wrong pair" trap), and some of which fit the sentence alone but have no partner. **Strategy: find the clue → predict the meaning → look for the matched pair → reject lone words and off-meaning pairs.**
>
> Each cluster: a **ready-made answer pair** (the two words that mean the same), **why they pair**, the **trigger sentence type** that summons this meaning, an **example stem**, and the **distractor traps** the GRE plants.

### 5.1 lavish praise → **extol / laud**
- **Why they pair:** both = praise highly; interchangeable in formal register.
- **Trigger:** a reviewer/critic/biographer reacting positively; contrast with expected criticism.
- **Stem:** "Far from panning the debut, critics began to ______ it as a masterpiece."
- **Distractor traps:** **deride / disparage** (a valid *pair* but opposite meaning); **acclaim** (fits alone but its partner **laud/extol** is the intended pair — beware three near-synonyms where only two are listed).

### 5.2 harsh criticism (speech) → **diatribe / tirade**
- **Why they pair:** both = a long, bitter verbal attack (nouns).
- **Trigger:** "delivered a ______," "launched into a ______"; an angry monologue.
- **Stem:** "What began as a measured review devolved into a ______ against modern art."
- **Distractor traps:** **eulogy / panegyric** (opposite pair); **harangue** (close, but pairs better as a *lecture*); **rebuke** (single act, no listed partner).

### 5.3 weaken / drain energy → **enervate / debilitate**
- **Why they pair:** both = to sap strength.
- **Trigger:** heat, illness, long effort leaving someone *too weak to act*.
- **Stem:** "The relentless humidity ______ the hikers, who could barely finish."
- **Distractor traps:** **invigorate / energize** (opposite, and **enervate** *looks* like "energize" — the #1 SE trap); **exhaust** (fits but its partner is enervate/debilitate).

### 5.4 grow rapidly → **burgeon / proliferate**
- **Why they pair:** both = increase/spread quickly.
- **Trigger:** populations, industries, ideas expanding; contrast with decline.
- **Stem:** "As the suburb grew, strip malls ______ along every highway."
- **Distractor traps:** **dwindle / wane** (opposite pair); **flourish** (fits alone).

### 5.5 lessen (pain/severity) → **mitigate / assuage**  (also alleviate, allay, mollify, palliate)
- **Why they pair:** both = make less severe / soothe.
- **Trigger:** grief, anger, pain, harshness being reduced.
- **Stem:** "A sincere apology did much to ______ her resentment."
- **Distractor traps:** **aggravate / exacerbate** (opposite pair); **mitigate** vs **mollify** — mollify is for *people/feelings*, mitigate for *conditions*; the GRE may force you to pick the pair that matches the object.

### 5.6 brief and to the point → **laconic / terse**  (also succinct, pithy)
- **Why they pair:** both = using few words.
- **Trigger:** a person/answer notable for shortness; contrast with verbosity.
- **Stem:** "Reporters found the senator frustratingly ______, offering only yes-or-no replies."
- **Distractor traps:** **garrulous / loquacious** (opposite pair); **curt** (fits but adds *rudeness*); **succinct/pithy** as the praise-flavored synonyms.

### 5.7 stubborn / won't compromise → **intransigent / obdurate**  (also obstinate, intractable)
- **Why they pair:** both = refusing to budge.
- **Trigger:** stalled negotiations, fixed positions.
- **Stem:** "Talks collapsed because the union remained ______ on pensions."
- **Distractor traps:** **amenable / tractable** (opposite pair); **adamant** (fits, pairs with obdurate); **pertinacious** (persistent, slightly different flavor).

### 5.8 plausible but false → **specious / spurious**  (reasoning: fallacious, sophistic)
- **Why they pair:** both = false despite a convincing surface.
- **Trigger:** an argument that "seems right" but isn't; debunking.
- **Stem:** "The theory's ______ logic collapsed under a single counterexample."
- **Distractor traps:** **sound / valid** (opposite); **spacious** (look-alike, wrong); **dubious** (doubtful, but not "deceptively plausible").

### 5.9 short-lived → **ephemeral / evanescent**  (also transient, fleeting)
- **Why they pair:** both = lasting a very short time.
- **Trigger:** fame, beauty, moods, fads that quickly vanish.
- **Stem:** "Internet fame is notoriously ______, forgotten within weeks."
- **Distractor traps:** **enduring / permanent** (opposite pair); **transient/fleeting** (valid alt pair — read which two are listed).

### 5.10 excessively talkative → **garrulous / loquacious**  (also voluble)
- **Why they pair:** both = chatty.
- **Stem:** "Normally reserved, after two cups of coffee he turned positively ______."
- **Distractor traps:** **taciturn / reticent** (opposite pair); **verbose/prolix** (wordy in *writing*, slightly different domain).

### 5.11 calm self-possession → **equanimity / aplomb**  (also composure, sangfroid)
- **Why they pair:** both = poise under pressure (nouns).
- **Stem:** "She fielded the hostile questions with remarkable ______."
- **Distractor traps:** **panic / agitation** (opposite); **indifference** (fits emotionally but means *not caring*, not *staying calm*).

### 5.12 stir up / incite → **foment / instigate**  (also kindle, provoke)
- **Why they pair:** both = to provoke (trouble, rebellion).
- **Stem:** "Agitators sought to ______ unrest among the workers."
- **Distractor traps:** **quell / quash / suppress** (opposite); **foment** vs **ferment** (look-alike — ferment can work figuratively too; watch spelling/sense).

### 5.13 praise-confirm a claim → **corroborate / substantiate**  (also validate, verify)
- **Why they pair:** both = support with evidence.
- **Stem:** "Independent labs ______ the original findings."
- **Distractor traps:** **refute / rebut** (opposite pair); **corroborate** vs **collaborate** (look-alike); **bolster** (supports, but of confidence/morale more than claims).

### 5.14 hard to understand → **abstruse / recondite**  (also esoteric, arcane)
- **Why they pair:** both = obscure because deep/specialized.
- **Stem:** "The lecture was so ______ that even graduate students were lost."
- **Distractor traps:** **lucid / accessible** (opposite); **obscure** (fits, but pairs with abstruse/recondite); **esoteric** adds *known to few* nuance.

### 5.15 wasteful generosity / lavish → **prodigal / profligate**  (spending)
- **Why they pair:** both = recklessly extravagant with money.
- **Stem:** "His ______ spending soon exhausted the family fortune."
- **Distractor traps:** **frugal / thrifty / parsimonious** (opposite); **prodigal** can also evoke "returning" (prodigal son) — context is *spending* here.

### 5.16 reduce in force gradually → **abate / subside**  (also wane, ebb, recede)
- **Why they pair:** both = die down.
- **Stem:** "Only by nightfall did the storm finally ______."
- **Distractor traps:** **intensify / surge** (opposite); **abate** (transitive *and* intransitive) vs **mitigate** (needs an object to lessen).

### 5.17 servile flatterer behavior → **obsequious / sycophantic**  (also fawning, unctuous)
- **Why they pair:** both = fawningly servile.
- **Stem:** "The ______ assistant agreed with every word the boss said."
- **Distractor traps:** **forthright / candid** (opposite); **fulsome** (excessive praise — close, but describes the *praise*, not the manner).

### 5.18 generous and noble → **magnanimous / munificent**
- **Why they pair:** both = big-hearted in giving/forgiving. (Munificent leans *money*; magnanimous leans *spirit/forgiveness* — pair them when the sentence stresses lavish generosity.)
- **Stem:** "In a ______ gesture, the victor praised his defeated rival."
- **Distractor traps:** **petty / vindictive** (opposite); **benevolent** (kindly, but milder).

### 5.19 cause to feel uneasy → **disconcert / perturb**  (also discompose, faze, fluster)
- **Why they pair:** both = unsettle, throw off.
- **Stem:** "The sudden silence ______ the speaker, who lost his place."
- **Distractor traps:** **reassure / soothe** (opposite); **disconcert** vs **disconnect** (look-alike).

### 5.20 openly bad / glaring → **flagrant / egregious**  (also blatant, glaring)
- **Why they pair:** both = conspicuously bad.
- **Stem:** "The referee missed a ______ foul in plain sight."
- **Distractor traps:** **trivial / negligible** (opposite); **flagrant** (of an offense) vs **fragrant** (smell — pure look-alike trap).

> **SE checklist:** ① underline the **clue** (the word/phrase that fixes the blank's meaning); ② predict in your own word; ③ scan for **two** options matching the prediction; ④ if you find *three* near-synonyms, recheck the clue's exact shade; ⑤ if you find a tidy pair that means the *opposite* of the clue, it's the trap — discard it. Cross-reference these pairs with §6 (polarity) to confirm direction before committing.

## 6. Text Completion Polarity Map

> **The single most powerful TC move is getting the *direction* right.** Before you worry about the exact word, decide whether the blank is **positive (+)**, **negative (−)**, or **neutral (0)**, and whether the sentence **continues** (same direction) or **reverses** (opposite direction). Signal words tell you which:
>
> - **Continuation / same direction:** *and, moreover, indeed, in fact, similarly, because, since, thus, consequently, so.*
> - **Reversal / opposite direction:** *but, yet, however, although, despite, while, nonetheless, on the contrary, far from, paradoxically, surprisingly, rather than.*
>
> Read the polarity, then pull the matching word bank below. Each map gives **+ words / − words**, the **GRE usage**, and the **TC logic** that connects clue → blank.

### 6.1 Master Positive ↔ Negative bank
| Polarity | Words |
|---|---|
| **+ (approval / good)** | laud, extol, acclaim, commend, esteem, venerate, sanguine, propitious, auspicious, salutary, magnanimous, munificent, judicious, perspicacious, lucid, equanimity, intrepid, probity |
| **− (disapproval / bad)** | castigate, excoriate, denounce, vilify, malign, deride, disparage, baleful, baneful, pernicious, deleterious, noxious, invidious, fatuous, specious, mendacity, craven, obdurate |
| **0 (neutral / factual)** | denote, comprise, constitute, delineate, enumerate, ostensible, putative, nominal, tangential, empirical |

- **TC logic:** if the clue is praise and the connector is *and/indeed*, the blank is **+**; if the connector is *but/although*, the blank flips to **−**. Eliminate every option on the wrong side *before* fine-tuning.

### 6.2 Praise (+) ↔ Criticism (−)
- **+ :** laud, extol, acclaim, eulogize, commend, exalt, lionize, tout.
- **− :** castigate, excoriate, lambaste, censure, vilify, malign, decry, disparage, denigrate, deprecate.
- **GRE usage:** reviews, biographies, historiography ("Once ______ as a hero, he is now ______ as a tyrant" → +, −).
- **TC logic:** *"Far from praising"* → flip to criticism; *"not only admired but"* → intensify praise.

### 6.3 Increase (+/0) ↔ Decrease (−/0)
- **up:** augment, burgeon, proliferate, amplify, escalate, accrue, wax, swell.
- **down:** abate, attenuate, diminish, dwindle, ebb, wane, subside, curtail, deplete, sap, enervate.
- **GRE usage:** trends, quantities, intensity. Watch contrast: *"Although production ______, profits ______"* (one up, one down).
- **TC logic:** "despite efforts to *boost*… it only ______" → a *down* word (enervate/dwindle).

### 6.4 Calm (+/0) ↔ Conflict / Agitation (−)
- **calm:** placate, mollify, assuage, pacify, conciliate, allay, soothe, temper; equanimity, composure, serene.
- **conflict:** foment, inflame, provoke, incite, exacerbate, roil, agitate; bellicose, belligerent, contentious, truculent, pugnacious.
- **GRE usage:** diplomacy, temperament. "Rather than ______ the crowd, his speech ______ it" → (calm) vs (inflame).
- **TC logic:** *provoke ↔ placate* is the canonical reversal pair; the connector decides which.

### 6.5 Clarity (+) ↔ Obscurity (−)
- **clear:** lucid, limpid, perspicuous, cogent, elucidate, transparent, intelligible.
- **obscure:** abstruse, recondite, esoteric, arcane, opaque, turgid, cryptic, nebulous, obfuscate.
- **GRE usage:** prose style, explanations. "Praised for clarity elsewhere, here he is maddeningly ______" → an *obscure* word.
- **TC logic:** if the writer is *criticized*, the blank for "his style" is **obscure**; if praised, **lucid**.

### 6.6 Certainty (+/0) ↔ Ambiguity / Doubt (−/0)
- **certain:** unequivocal, categorical, incontrovertible, emphatic, definitive, unqualified.
- **ambiguous:** equivocal, ambiguous, ambivalent, tentative, provisional, nebulous, vacillating.
- **GRE usage:** statements, evidence, attitudes. "His ______ answer left both sides claiming victory" → *equivocal/ambiguous*.
- **TC logic:** *"refused to commit"* → ambiguity bank; *"left no room for doubt"* → certainty bank.

### 6.7 Sincerity / Frankness (+) ↔ Deception / Pretense (−)
- **+ :** candid, forthright, ingenuous, guileless, frank, aboveboard, veracious.
- **− :** disingenuous, dissemble, duplicitous, specious, mendacious, evasive, cagey, sophistical.
- **GRE usage:** character, testimony. "Beneath his ______ charm lay a ______ schemer" → (ingenuous-seeming) vs (duplicitous).
- **TC logic:** look for *"seemingly… but actually"* — first blank +, second −.

### 6.8 Praise-worthy thrift (+) ↔ Stinginess (−) ↔ Wastefulness (−)
- **virtuous thrift (+/0):** frugal, abstemious, economical, prudent, austere.
- **stingy (−):** parsimonious, miserly, niggardly, penurious, tightfisted.
- **wasteful (−):** prodigal, profligate, extravagant, improvident, spendthrift.
- **GRE usage:** money/character. Note the **three-way** split — the connector and tone decide whether thrift is admired (frugal) or condemned (miserly).
- **TC logic:** *"so ______ he reused tea bags"* leans **miserly** if mocked, **frugal** if praised.

### 6.9 Common single-blank reversers (memorize the flip)
| Clue says… | …and connector flips it, so blank = |
|---|---|
| generous | **parsimonious / niggardly** |
| energize | **enervate / sap** |
| praise | **castigate / excoriate** |
| clear | **abstruse / opaque** |
| calm | **foment / inflame** |
| honest | **duplicitous / mendacious** |
| permanent | **ephemeral / transient** |
| humble | **supercilious / haughty** |
| flexible | **intransigent / obdurate** |
| brave | **craven / pusillanimous** |

> **TC discipline for 2- and 3-blank questions:** solve the blank you're *most sure of first* (usually the one with the clearest signal word), lock its polarity, then use sentence logic to constrain the rest. Never pick a word you love before checking its **direction** against the connector — a perfect-meaning word on the wrong side of a *but* is always wrong.

## 7. Common Confusing Words

> All **106** confusable pairs/groups from the source, transformed into study material. Format per item: each word with its **faithful meaning**, then a bolded **GRE distinction / memory trick** (and the common mistake it prevents). These are prime **Text Completion** material — the GRE plants the look-alike as a distractor and rewards the precise meaning.

**1. **acrid** vs **acrimonious****  
&nbsp;&nbsp;• *acrid* — sharp, bitterly pungent (acrid smell of burning hair)  
&nbsp;&nbsp;• *acrimonious* — stinging, caustic in tone (an acrimonious argument)  
&nbsp;&nbsp;↳ **acrid** = a *smell/taste*; **acrimonious** = a *dispute/manner*. Trick: acrimon**ious** people argue.

**2. **affection** vs **affectation****  
&nbsp;&nbsp;• *affection* — feeling of love  
&nbsp;&nbsp;• *affectation* — an artificial show, insincere pretense  
&nbsp;&nbsp;↳ **-ation** = a put-on. Mistake: writing 'affectation' for genuine fondness.

**3. **allude** vs **elude****  
&nbsp;&nbsp;• *allude* — to make an indirect reference  
&nbsp;&nbsp;• *elude* — to escape, evade  
&nbsp;&nbsp;↳ **a**llude = **a**lludes to an idea; **e**lude = **e**scape. GRE pairs 'allude/allusion' with hints.

**4. **amiable** vs **amicable** vs **amenable****  
&nbsp;&nbsp;• *amiable* — lovable, agreeable — of *persons*  
&nbsp;&nbsp;• *amicable* — peaceable, harmonious — of *arrangements/settlements*  
&nbsp;&nbsp;• *amenable* — readily managed, willing to be led  
&nbsp;&nbsp;↳ amiable PERSON, amicable SETTLEMENT, amenable = persuadable.

**5. **appraise** vs **apprise****  
&nbsp;&nbsp;• *appraise* — to estimate, judge (performance appraisal)  
&nbsp;&nbsp;• *apprise* — to inform (they apprised the police)  
&nbsp;&nbsp;↳ appr**ai**se = put a price; appr**i**se = give info.

**6. **alternate** vs **alternative****  
&nbsp;&nbsp;• *alternate* — every other (visits on alternate days)  
&nbsp;&nbsp;• *alternative* — a choice between two or more options  
&nbsp;&nbsp;↳ alternate = takes turns; alternative = an option.

**7. **attenuate** vs **extenuate****  
&nbsp;&nbsp;• *attenuate* — reduce, weaken (attenuated bodies)  
&nbsp;&nbsp;• *extenuate* — lessen the magnitude of guilt (nothing can extenuate your conduct)  
&nbsp;&nbsp;↳ extenuate almost always pairs with *circumstances/guilt*.

**8. **baneful** vs **baleful****  
&nbsp;&nbsp;• *baneful* — ruinous, poisonous (the bane of his life)  
&nbsp;&nbsp;• *baleful* — deadly, destructive, sinister (baleful looks)  
&nbsp;&nbsp;↳ baleful = a *menacing look*; baneful = *causes ruin*.

**9. **brusque** vs **burlesque****  
&nbsp;&nbsp;• *brusque* — blunt, abrupt (brusque manner)  
&nbsp;&nbsp;• *burlesque* — an imitation that ridicules  
&nbsp;&nbsp;↳ brusque = curt; burlesque = parody.

**10. **beneficial** vs **beneficent****  
&nbsp;&nbsp;• *beneficial* — useful (advice proved beneficial)  
&nbsp;&nbsp;• *beneficent* — kind (a beneficent act)  
&nbsp;&nbsp;↳ benefi**ci**al = does good *to you*; benefi**ce**nt = *kindly person/act*.

**11. **bridal** vs **bridle****  
&nbsp;&nbsp;• *bridal* — of a wedding/bride  
&nbsp;&nbsp;• *bridle* — to control, check (bridle your passions)  
&nbsp;&nbsp;↳ bri**dle** = the horse's *bridle* restrains.

**12. **broach** vs **brooch****  
&nbsp;&nbsp;• *broach* — to initiate, open (broach the topic)  
&nbsp;&nbsp;• *brooch* — an ornament  
&nbsp;&nbsp;↳ broo**ch** = jewelry; broa**ch** = bring up.

**13. **canker** vs **canter****  
&nbsp;&nbsp;• *canker* — any ulcerous sore, cancer  
&nbsp;&nbsp;• *canter* — slow gallop  
&nbsp;&nbsp;↳ cant**er** = a horse's gait.

**14. **cannon** vs **canon****  
&nbsp;&nbsp;• *cannon* — big gun  
&nbsp;&nbsp;• *canon* — law (canons of justice)  
&nbsp;&nbsp;↳ one-N canon = rule/body of works.

**15. **capricious** vs **captious****  
&nbsp;&nbsp;• *capricious* — whimsical, unpredictable  
&nbsp;&nbsp;• *captious* — fault-finding (captious mother-in-law)  
&nbsp;&nbsp;↳ capTious = picks faults; capRicious = on a whim.

**16. **censor** vs **censure** vs **cynosure****  
&nbsp;&nbsp;• *censor* — to suppress, forbid, delete (film censors)  
&nbsp;&nbsp;• *censure* — rebuke, criticize adversely  
&nbsp;&nbsp;• *cynosure* — focal point of attraction  
&nbsp;&nbsp;↳ censor SUPPRESSES, censure SCOLDS, cynosure = center of attention.

**17. **cessation** vs **cession****

## 8. Word Groups Rebuilt for Memory

> Every thematic **Word Group** from the source, reorganized for active study. For each: the **shared theme**, the full **member list** (faithful to the source), any **antonym sub-group**, the single best **SE pairing** to memorize, and a **hook / distinction**. ⚠ flags the traps. These are your SE/TC raw material — learn each group as one chunk, then drill its antonym group back-to-back.

### HARSH / BITING / SHARP
*of speech or wit — cutting.*  
**Words:** acerbic, acrimonious, caustic, incisive, scathing, trenchant.  
**Best SE pair:** caustic ≈ scathing.  
**Hook:** All describe a *tongue*: trenchant/incisive cut cleanly; acrimonious adds bitterness.

### NOT THOROUGH
*done superficially.*  
**Words:** cursory, perfunctory, summarily, superficial.  
**Best SE pair:** cursory ≈ perfunctory.  
**Hook:** A *cursor* skims the surface → cursory glance, perfunctory effort.

### HINT / INDIRECT REFERENCE
*saying without stating.*  
**Words:** allude/allusion, cue, implicit, innuendo, insinuate, tacit.  
**Antonyms:** blatant, explicit, patent.  
**Best SE pair:** implicit ≈ tacit.  
**Hook:** Tacit = unspoken-but-understood; antonyms shout it (blatant/patent).

### CONFIRM / SUPPORT (a claim)
*back with evidence.*  
**Words:** affirm, clinch, corroborate, endorse, ratify, substantiate, testify, validate.  
**Antonyms:** unfounded.  
**Best SE pair:** corroborate ≈ substantiate.  
**Hook:** Evidence words — see §5.13. Antonym 'unfounded' = unsupported.

### STRENGTHEN / SUPPORT (a thing)
*prop up, reinforce.*  
**Words:** bastion, bolster, buttress, fortify, prime, prop.  
**Best SE pair:** bolster ≈ buttress.  
**Hook:** A buttress braces a wall; bolster braces morale.

### VARIETY
*made of many kinds.*  
**Words:** diversity, eclectic, motley, multifaceted, multifarious, sundry, variegate, versatile.  
**Best SE pair:** multifarious ≈ sundry.  
**Hook:** Eclectic = drawn from many sources; motley = mismatched mix.

### BAN
*officially forbid.*  
**Words:** interdict, sanction, veto.  
**Best SE pair:** interdict ≈ veto.  
**Hook:** ⚠ **sanction** is auto-antonym: here it = *forbid*, elsewhere = *approve*.

**Best SE pair:** deride ≈ mock.  
**Hook:** lampoon/satire = mock in writing; disdain/scorn/spurn = reject contemptuously.

### PRAISE
*express high approval.*  
**Words:** acclaim, accolade, applaud, approbation, commendable, eulogy, exalt, extol, glorify, hail, laud, lionize, plaudit.  
**Best SE pair:** laud ≈ extol.  
**Hook:** See §4.1 & §5.1. Direct antonyms = the DISAPPROVE / MOCK groups.

### AVOID / DODGE
*evade.*  
**Words:** avert, circumvent, deflect, elude, evade, evasive, parry, quibble, shirk, shunt, skirt, swerve.  
**Antonyms:** head-on.  
**Best SE pair:** elude ≈ evade.  
**Hook:** circumvent = get around (a rule); shirk = dodge a duty.

### NOT SERIOUS / FRIVOLOUS
*light, carefree.*  
**Words:** blithe, facetious, flippant, frivolous, frivolity, insouciant, levity.  
**Antonyms:** grave, gravity.  
**Best SE pair:** flippant ≈ facetious.  
**Hook:** levity = lack of seriousness; insouciant = carefree. Antonyms grave/gravity.

### HORRIFYING / SHOCKING
*dreadful.*  
**Words:** aghast, appalling, egregious, ghastly, grisly, gross, grotesque, gruesome, heinous, hideous, horrendous, monstrous, morbid, unsightly, vile.  
**Best SE pair:** heinous ≈ monstrous.  
**Hook:** egregious = outrageously bad; grisly/gruesome = causing horror.

## 10. Spaced Revision Plan

> **Why spacing works:** memories decay on a predictable curve; each successful recall *just before* you'd forget resets the curve flatter. The schedule below front-loads new words, then stretches the gaps. Pair it with the §9 flashcards and the §11 glossary.

### 10.1 The expanding-interval schedule (per batch of new words)
Learn words in **batches of 25** (one source wordlist, or one §8 group). For each batch, review on:

| Review | When | What to do |
|---|---|---|
| R1 | **Day 0** (learn) | Read meaning + hook + one example; write the word once. |
| R2 | **Day 1** | Active recall (Deck D): word → meaning, then meaning → word. |
| R3 | **Day 2** | Recall + place each word in its §4/§8 cluster. |
| R4 | **Day 4** | Recall only the ones you missed; add to "leeches" list. |
| R5 | **Day 7** | Mixed test with other batches; do 5 SE-style clues (§5). |
| R6 | **Day 14** | Recall + write a sentence using the word. |
| R7 | **Day 30** | Final pass; anything still shaky → permanent leech deck. |

A word that survives all 7 reviews without error is "retired" to monthly maintenance.

### 10.2 30-day sprint (all 31 wordlists + groups)
- **Days 1–16:** learn **2 wordlists/day** (50 words) at R1; each prior batch rides its R2–R7 schedule.
- **Days 17–22:** no new words — drill §7 (confusing pairs) and §8 (groups), ~18 pairs + 16 groups/day.
- **Days 23–27:** polarity & SE practice (§5/§6): 20 mixed TC/SE items/day; re-test weak clusters.
- **Days 28–30:** full glossary sweep (§11) at speed — flag every word you can't define in 2 seconds into the leech deck; hammer the leech deck.

### 10.3 Weekly template (steady, non-sprint pace)
| Day | Focus |
|---|---|
| Mon | New batch (25) — R1 + R2 of last week's batches |
| Tue | New batch (25) — R2/R3 cycles |
| Wed | Confusing pairs (§7) + roots (§3) |
| Thu | New batch (25) — R3/R4 cycles |
| Fri | Clusters (§4) + SE pairs (§5) practice |
| Sat | Polarity drills (§6) + 30 mixed TC/SE questions |
| Sun | **Leech day** — only the words you keep missing |

### 10.4 "Words I keep forgetting" (leech tracker)
Keep a running list here. A word earns a tally each time you miss it; at **3 tallies**, give it a custom hook and a sentence of your own.

| Word | Misses | My hook | My sentence |

| _e.g._ enervate | III | "cuts my NERVE" | The heat enervated me. |
